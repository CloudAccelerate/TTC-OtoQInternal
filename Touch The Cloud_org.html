About:


Oracle has invested in delivering a plethora of new Cloud Services and we want to show our customers how simple – yet powerful they are. We want to showcase their simplicity for the IT and for the non-IT community alike. We want to demonstrate how powerful they are, and how they should be considered as tools that can help an IT organization take the right steps in transforming for the Digital era.

We believe the best way to do this is to let our customers develop a real world business solution using our cloud services, hands-on!

Therefore, we have prepared an event where we can give customers access to a number of Oracle Cloud Services, whilst we have Oracle experts guide them in using our tools. Customers browse screens, configure integrations, develop processes and create a mobile applications as part of a guided “Touch The Cloud experience”.

What is this event about?

  • Allowing the customers to get an appreciation of Oracle’s cloud services by “touching” our services by following some very basic hands-on labs

  • This is note a training activity, i.e we do not expect that customers will go away knowing how to do integration etc - they just need to get an understanding and appreciation of Oracle's low code high productivity cloud services

Services Included:

Cloud Services covered in this event are:

» Integration Cloud Service

» Process Cloud Service

» Mobile Cloud Service

» Mobile Accelerator

Organizing Such an Event:


We have pre-packaged all the material and Cloud accounts required so other than arranging the logistics and inviting the customers you are good to go!

What is available:

  • Methodology for running your event

  • Documentation for the labs the customers will need to follow - these are easy to use

  • ICS, PCS, MCS and MAX environments to run the labs on

  • Marketing material (brochure, flyer)

  • A master presentation to be used to drive the day.

Target:

  • Customer

    • Having a strategic apps modernization goal

    • E-business Suite install base

    • SaaS install base

  • Audience:

    • Business Analysts that liaise with IT

    • Apps-IT Developers

    • Architects

    • Developers

  • No expertise or experience required.



Who delivers the workshop?

Facilitation of the event is provided by Oracle NAS Platform specialists. Each event is aimed at enabling the local sales and sales consulting team to replicate the approach as they wish.

What is needed from YOU:

  • To schedule the event, identify and invite the customers

  • Let the Platform specialist team know when are where this is looking to take place

  • To source a location with good internet connectivity. It could be hosted in an Oracle office or another location (for example a partner office, customer premises)

  • To provision catering for the day

What customers need to bring:

  • A laptop and a mobile phone (iOS or Android)- that's it!
